# System Requirements

## Functional Requirements
- **FR-1**: Adopters must fill out a 5-step lifestyle questionnaire.
- **FR-2**: Live compatibility scoring compared with cat personality vectors.
- **FR-3**: Video/Image behavior upload to execute OpenCV keyframe extraction.
- **FR-4**: Gemini API calls to parse behavior, mood, and recommendations.

## Non-Functional Requirements
- **NFR-1**: Page renders under 500ms.
- **NFR-2**: Sub-second ORM queries with indexing.
- **NFR-3**: CORS restrictions and input sanitization.
