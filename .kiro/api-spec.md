# API Specifications

- **POST `/auth/signup`**: Registers adopter or shelter profile.
- **POST `/auth/login`**: Authenticates user profile.
- **GET `/cats/`**: Lists cats with compatibility percentage.
- **POST `/questionnaire`**: Ingests adopter preferences.
- **POST `/behaviour-analysis`**: Accepts image/video files for AI mood parsing.
- **GET `/dashboard`**: Consolidated dashboard payload.
